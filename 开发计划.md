# MiniHabits MVP 开发计划

## 项目概述
基于《微习惯》理论的极简习惯养成应用，通过"小得不可思议"的目标设定，帮助用户无压力地建立持久的好习惯。

## 技术栈
- SwiftUI + SwiftData
- MVVM 架构
- 遵循 SOLID 原则
- 支持 iOS 系统

## 开发原则
1. **每个阶段独立完成**：确保每个功能模块完成后能编译成功
2. **增量式开发**：每完成一个阶段都有可用的功能增量
3. **用户测试驱动**：每个阶段完成后等待用户测试确认
4. **代码质量保证**：遵循SOLID原则和Swift最佳实践
5. **文档同步更新**：随着开发进度更新本文档
